# 注塑机数字孪生系统 - 飞渡 AirCity Web 平台迁移方案

## 一、迁移概述

### 1.1 源平台（当前）
- **引擎**: Unreal Engine 4.26
- **语言**: C++ (659行核心代码)
- **运行环境**: Windows 桌面应用
- **通信**: MqttUtilities 插件

### 1.2 目标平台（飞渡）
- **平台**: AirCity Web 平台
- **技术栈**: WebGL + JavaScript/TypeScript
- **运行环境**: 浏览器端
- **通信**: MQTT over WebSocket

### 1.3 迁移范围
✅ MQTT 通信（14+传感器字段）
✅ 双向控制（启停机器、参数设置）
✅ 部件级控制（26个组件动画）
✅ 多设备支持（10+并发设备）
✅ 实时 HUD 显示

### 3.1 MQTT 通信层

#### UE4 实现（C++）
```cpp
// FastBeeMqttSubsystem.cpp
void UFastBeeMqttSubsystem::OnMessageReceived(const FMqttMessage& Message) {
    TSharedPtr<FJsonObject> JsonObject;
    TSharedRef<TJsonReader<>> Reader = TJsonReaderFactory<>::Create(Message.GetPayloadAsString());
    if (FJsonSerializer::Deserialize(Reader, JsonObject)) {
        FString DeviceId = JsonObject->GetStringField(TEXT("deviceId"));
        float Rpm = JsonObject->GetNumberField(TEXT("rpm"));
        // ...
    }
}
```

#### AirCity Web 实现（JavaScript）
```javascript
// mqtt-client.js
import mqtt from 'mqtt';

class MqttClient {
    constructor(config) {
        this.client = mqtt.connect(`ws://${config.host}:${config.wsPort}`, {
            clientId: config.clientId
        });

        this.client.on('message', (topic, payload) => {
            const data = JSON.parse(payload.toString());
            this.handleMessage(topic, data);
        });
    }

    handleMessage(topic, data) {
        const deviceId = data.deviceId;
        const rpm = data.rpm || 0;
        const running = data.running || false;
        // 触发事件通知 3D 场景更新
        this.emit('telemetry', { deviceId, rpm, running, ...data });
    }

    publish(topic, payload) {
        this.client.publish(topic, JSON.stringify(payload));
    }
}
```

**关键差异**:
- UE4 使用 TCP 1883 端口，Web 必须使用 WebSocket（通常 8083/9001 端口）
- 需要配置 MQTT Broker 支持 WebSocket（EMQX 默认支持）

### 3.2 部件级控制

#### UE4 实现（C++）
```cpp
// MachinePartController.cpp
void UMachinePartController::UpdateScrewRotation(float Rpm, float DeltaTime) {
    if (ScrewComponent) {
        float RotationSpeed = Rpm * 6.0f; // RPM to degrees/sec
        FRotator NewRotation = ScrewComponent->GetRelativeRotation();
        NewRotation.Yaw += RotationSpeed * DeltaTime;
        ScrewComponent->SetRelativeRotation(NewRotation);
    }
}
```

#### AirCity Web 实现（JavaScript）
```javascript
// machine-controller.js
class MachineController {
    constructor(machineModel) {
        this.screwParts = [
            machineModel.getObjectByName('Screw_01'),
            machineModel.getObjectByName('Screw_02'),
            // ... 6个螺杆部件
        ];
    }

    updateScrewRotation(rpm, deltaTime) {
        const rotationSpeed = rpm * 6.0 * (Math.PI / 180); // RPM to radians/sec
        this.screwParts.forEach(screw => {
            if (screw) {
                screw.rotation.y += rotationSpeed * deltaTime;
            }
        });
    }

    updateScrewPosition(position) {
        this.screwParts.forEach(screw => {
            if (screw) {
                screw.position.x = position; // 轴向移动
            }
        });
    }
}
```

**26个组件映射表**:
| 组件类型 | 数量 | 命名规则 | 控制参数 |
|---------|------|---------|---------|
| 螺杆旋转 | 6 | Screw_01~06 | rpm (旋转速度) |
| 螺杆轴向 | 6 | Screw_01~06 | screwPosition (X轴位移) |
| 注射单元 | 14 | InjectionUnit_01~14 | injectionUnitPosition (Z轴位移) |
| 料筒温度 | 3 | Barrel_01~03 | barrelTemperature (材质颜色) |
| 模具温度 | 3 | Mold_01~03 | moldTemperature (材质颜色) |

### 3.3 多设备支持

#### UE4 实现（C++）
```cpp
// FastBeeMqttSubsystem.cpp - 3级Actor映射策略
AActor* UFastBeeMqttSubsystem::FindActorForDevice(const FString& DeviceId) {
    // Level 1: 精确Tag匹配
    TArray<AActor*> FoundActors;
    UGameplayStatics::GetAllActorsWithTag(GetWorld(), FName(*DeviceId), FoundActors);
    if (FoundActors.Num() > 0) return FoundActors[0];

    // Level 2: 名称匹配
    UGameplayStatics::GetAllActorsOfClass(GetWorld(), AActor::StaticClass(), FoundActors);
    for (AActor* Actor : FoundActors) {
        if (Actor->GetName().Contains(DeviceId)) return Actor;
    }

    // Level 3: 类关键字匹配
    UGameplayStatics::GetAllActorsOfClass(GetWorld(), AActor::StaticClass(), FoundActors);
    for (AActor* Actor : FoundActors) {
        if (Actor->GetClass()->GetName().Contains(Settings->MachineClassKeyword)) {
            return Actor;
        }
    }
    return nullptr;
}
```

#### AirCity Web 实现（JavaScript）
```javascript
// device-manager.js
class DeviceManager {
    constructor(scene) {
        this.scene = scene;
        this.deviceMap = new Map(); // deviceId -> MachineController
    }

    findActorForDevice(deviceId) {
        // Level 1: userData.deviceId 精确匹配
        let actor = this.scene.getObjectByProperty('userData.deviceId', deviceId);
        if (actor) return actor;

        // Level 2: 名称匹配
        actor = this.scene.getObjectByName(deviceId);
        if (actor) return actor;

        // Level 3: 类型关键字匹配（userData.type === 'BP_Machine'）
        this.scene.traverse(obj => {
            if (obj.userData.type === 'BP_Machine' && !this.deviceMap.has(obj.uuid)) {
                actor = obj;
                return true; // 停止遍历
            }
        });

        return actor;
    }

    registerDevice(deviceId, telemetryData) {
        if (!this.deviceMap.has(deviceId)) {
            const actor = this.findActorForDevice(deviceId);
            if (actor) {
                const controller = new MachineController(actor);
                this.deviceMap.set(deviceId, controller);
            }
        }

        const controller = this.deviceMap.get(deviceId);
        if (controller) {
            controller.update(telemetryData);
        }
    }
}
```

### 3.5 双向控制

#### UE4 实现（C++）
```cpp
// FastBeeMqttSubsystem.cpp
void UFastBeeMqttSubsystem::StartMachine(const FString& DeviceId) {
    FString Topic = FString::Printf(TEXT("fastbee/%s/command"), *DeviceId);
    TSharedPtr<FJsonObject> JsonObject = MakeShareable(new FJsonObject);
    JsonObject->SetStringField(TEXT("command"), TEXT("start"));
    JsonObject->SetStringField(TEXT("deviceId"), DeviceId);

    FString OutputString;
    TSharedRef<TJsonWriter<>> Writer = TJsonWriterFactory<>::Create(&OutputString);
    FJsonSerializer::Serialize(JsonObject.ToSharedRef(), Writer);

    MqttClient->Publish(Topic, OutputString);
}
```

#### AirCity Web 实现（JavaScript）
```javascript
// control-panel.js
class ControlPanel {
    constructor(mqttClient) {
        this.mqttClient = mqttClient;
        this.setupUI();
    }

    setupUI() {
        document.getElementById('btn-start').addEventListener('click', () => {
            const deviceId = document.getElementById('device-select').value;
            this.startMachine(deviceId);
        });

        document.getElementById('btn-stop').addEventListener('click', () => {
            const deviceId = document.getElementById('device-select').value;
            this.stopMachine(deviceId);
        });
    }

    startMachine(deviceId) {
        const topic = `fastbee/${deviceId}/command`;
        const payload = {
            command: 'start',
            deviceId: deviceId,
            timestamp: Date.now()
        };
        this.mqttClient.publish(topic, payload);
    }

    stopMachine(deviceId) {
        const topic = `fastbee/${deviceId}/command`;
        const payload = {
            command: 'stop',
            deviceId: deviceId,
            timestamp: Date.now()
        };
        this.mqttClient.publish(topic, payload);
    }

    setParameter(deviceId, paramName, paramValue) {
        const topic = `fastbee/${deviceId}/command`;
        const payload = {
            command: 'setParameter',
            deviceId: deviceId,
            parameter: paramName,
            value: paramValue,
            timestamp: Date.now()
        };
        this.mqttClient.publish(topic, payload);
    }
}
```

## 六、3D模型迁移

### 6.1 从UE4导出模型

**方法1: 使用UE4内置导出**
1. 打开 `Content/Project_machine/MainMachinery/BP_Machine.uasset`
2. 右键 → Export → 选择 FBX 格式
3. 使用 Blender 转换为 GLTF/GLB 格式

**方法2: 使用Datasmith插件**
1. 安装 Datasmith Exporter for UE4
2. 导出整个场景为 Datasmith 格式
3. 使用 Datasmith SDK 转换为 Web 格式

**命名约定保持一致**:
- UE4: `Screw_01`, `InjectionUnit_01`, `Barrel_01`
- GLTF: 保持相同命名，确保 JavaScript 代码能正确查找

### 6.2 模型优化

| 优化项 | UE4 | Web (目标) |
|--------|-----|-----------|
| 多边形数 | 不限 | < 50K 三角面 |
| 纹理分辨率 | 2048x2048 | 1024x1024 |
| 材质数量 | 不限 | < 10 个 |
| 文件大小 | 不限 | < 5MB (压缩后) |

## 十、风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 模型导出失败 | 无法显示3D | 使用简化模型或基础几何体替代 |
| WebSocket不支持 | MQTT无法连接 | 确认EMQX配置，使用8083端口 |
| 性能不足 | FPS < 30 | 降低模型精度，减少实时更新频率 |
| 浏览器兼容性 | 部分浏览器无法运行 | 使用Polyfill，降级到Canvas 2D |
| 时间不足 | 无法完成全部功能 | 优先实现P0功能（MQTT+3D+HUD） |
