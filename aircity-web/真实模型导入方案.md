# 真实 3D 模型导入方案

## 问题分析

UE4 的 `.uasset` 文件是专有格式，无法直接在 Web 中使用。需要导出为 Web 兼容格式（GLTF/GLB）。

---

## 方案 1: 使用 UE4 导出 GLTF（推荐）

### 步骤 1: 安装 GLTF 导出插件

1. 打开 UE4 项目 `AirCityExplorer.uproject`
2. Edit → Plugins
3. 搜索 "GLTF"
4. 启用 "glTF Exporter" 插件
5. 重启编辑器

### 步骤 2: 导出 BP_Machine

1. 在 Content Browser 中找到：
   ```
   Content/Project_machine/MainMachinery/BP_Machine
   ```

2. 右键点击 BP_Machine → Asset Actions → Export

3. 选择导出格式：
   - **格式**: glTF Binary (*.glb) 或 glTF (*.gltf)
   - **推荐**: GLB（单文件，包含所有资源）

4. 保存位置：
   ```
   C:\Users\24045\Desktop\Uproject\aircity-web\public\models\machine.glb
   ```

### 步骤 3: 确保组件名称保留

导出时勾选：
- ☑ Export Preview Mesh
- ☑ Export All LODs
- ☑ Export Vertex Colors
- ☑ Export Materials
- ☑ **Export Mesh Data** (重要：保留组件层级)

---

## 方案 2: 使用 Datasmith 导出

如果 GLTF 导出插件不可用：

1. File → Datasmith → Export Scene
2. 选择 BP_Machine
3. 导出为 `.udatasmith`
4. 使用 Blender 的 Datasmith 插件导入
5. 在 Blender 中导出为 GLTF

### 使用 Blender 转换

1. 安装 Blender: https://www.blender.org/download/
2. 导入原始模型（FBX/OBJ）
3. 确保组件命名正确：
   ```
   mechine_1_Zhushezhen
   mechine_1_Zhushezhen_001
   mechine_1_Zhushezhen_002
   ...
   ```
4. File → Export → glTF 2.0 (.glb)
5. 导出设置：
   - Format: glTF Binary (.glb)
   - Include: Selected Objects
   - Transform: +Y Up
   - Geometry: Apply Modifiers

## 方案 4: 使用 UE4 Python 脚本批量导出

创建 Python 脚本在 UE4 中运行：

```python
# export_gltf.py
import unreal

# 获取 BP_Machine 资源
asset_path = "/Game/Project_machine/MainMachinery/BP_Machine"
asset = unreal.load_asset(asset_path)

# 设置导出选项
export_task = unreal.AssetExportTask()
export_task.object = asset
export_task.filename = "C:/Users/24045/Desktop/Uproject/aircity-web/public/models/machine.glb"
export_task.automated = True
export_task.replace_identical = True

# 执行导出
unreal.Exporter.run_asset_export_task(export_task)
print("导出完成！")
```

运行方式：
1. UE4 Editor → Window → Developer Tools → Python Console
2. 粘贴并运行脚本

## 如果组件名称丢失

1. 导入 GLB 到 Blender
2. 在 Outliner 中找到对应的 Mesh
3. 按照 COMPONENTS.txt 中的名称重命名
4. 重新导出 GLB
