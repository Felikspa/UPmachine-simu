## 解决方案：导出真实模型

### 方案 1: UE4 导出 GLTF（推荐）

#### 步骤 1: 安装插件
1. 打开 UE4 项目 `AirCityExplorer.uproject`
2. Edit → Plugins
3. 搜索 "GLTF"
4. 启用 "glTF Exporter" 插件
5. 重启编辑器

#### 步骤 2: 导出模型
1. 在 Content Browser 中找到：
   ```
   Content/Project_machine/MainMachinery/BP_Machine
   ```
2. 右键 → Asset Actions → Export
3. 选择格式：**glTF Binary (*.glb)**
4. 保存到：
   ```
   C:\Users\24045\Desktop\Uproject\aircity-web\public\models\machine.glb
   ```

#### 步骤 3: 更新代码
导出完成后，我会立即更新代码加载真实模型。

---

### 方案 2: 使用 Datasmith

如果 GLTF 插件不可用：

1. File → Datasmith → Export Scene
2. 选择 BP_Machine
3. 导出为 `.udatasmith`
4. 使用 Blender 的 Datasmith 插件导入
5. 在 Blender 中导出为 GLTF

---

### 方案 3: 使用 UE4 Python 脚本

在 UE4 中运行 Python 脚本自动导出：

```python
import unreal

# 获取 BP_Machine
asset_path = "/Game/Project_machine/MainMachinery/BP_Machine"
asset = unreal.load_asset(asset_path)

# 导出为 GLTF
export_task = unreal.AssetExportTask()
export_task.object = asset
export_task.filename = "C:/Users/24045/Desktop/Uproject/aircity-web/public/models/machine.glb"
export_task.automated = True
export_task.replace_identical = True

unreal.Exporter.run_asset_export_task(export_task)
print("导出完成！")
```

运行方式：
- UE4 Editor → Window → Developer Tools → Python Console
- 粘贴并运行

我会修改 `model-generator.js`，从加载占位几何体改为加载真实 GLB 文件：

```javascript
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader.js';

export function loadRealMachineModel() {
    return new Promise((resolve, reject) => {
        const loader = new GLTFLoader();
        loader.load('/models/machine.glb', (gltf) => {
            const machineGroup = gltf.scene;
            machineGroup.name = 'machine01';
            machineGroup.userData = {
                type: 'BP_Machine',
                deviceId: 'machine01'
            };
            resolve(machineGroup);
        }, undefined, reject);
    });
}
```
