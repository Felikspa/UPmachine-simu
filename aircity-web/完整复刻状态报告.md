## 🎯 下一步操作

### 1. 重启 Mosquitto（必须）


验证成功：
```bash
netstat -an | findstr "8083"
# 应该看到: TCP    0.0.0.0:8083
```

### 2. 刷新浏览器

访问: http://localhost:5174

检查左下角连接状态：
- ❌ 未连接 → 需要重启 Mosquitto
- ✅ 已连接 → 配置成功

### 3. 测试功能

#### 查看实时数据
- HUD 显示设备状态
- 初始状态: 停止 (RPM=0)

#### 启动机器
1. 点击右侧 **"启动机器"** 按钮
2. 观察:
   - 运行状态变为 "运行中" (绿色)
   - RPM 开始变化 (1000-1500)
   - 螺杆开始旋转
   - 温度上升

#### 停止机器
1. 点击 **"停止机器"** 按钮
2. 观察:
   - 运行状态变为 "停止" (红色)
   - RPM 归零
   - 螺杆停止旋转

#### 设置转速
1. 输入框输入转速 (例如: 1800)
2. 点击 **"设置"** 按钮
3. 观察 RPM 变化
